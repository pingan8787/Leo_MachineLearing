## 1.TensirFlow随机数生成函数
tf.random_normal   # 随机数分布：正态分布 ； 主要参数：平均值、标准差、取值类型
tf.trucated_normal # 随机数分布：正态分布(若生成的随机数偏离平均值超过2个标准差，则重新随机)
                   # 主要参数：平均值、标准差、取值范围
tf.random_uniform  # 随机数分布：平均分布 ；主要参数：最大 最小取值、取值类型
tf.random_gamma    # 随机数分布：Gamma分布 ； 主要参数：形状参数alpha、尺度参数beta、取值类型。

## 2.TensorFlow常数生成函数
tf.zeros    # 功能：产生全0的数组 ； 样例：tf.zeros([2,3],int32) -> [[0,0,0],[0,0,0]]
tf.ones     # 功能：产生全1的数组 ； 样例：tf.ones([2,3],int32) -> [[1,1,1],[1,1,1]]
tf.fill     # 功能：产生一个全部为给定数字的数组 ； 样例：tf.fill([2,3],9)->[[9,9,9],[9,9,9]]
tf.constant # 功能：产生一个给定值的常量 ；样例：tf.constant([1,2,3]) -> [1,2,3]
